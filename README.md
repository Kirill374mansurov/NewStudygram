## Описание

**Studygram** — платформа для поиска, публикации и сохранения учебных материалов. Пользователи могут делиться полезными ресурсами, распределять их по темам, собирать подборки и формировать собственный план обучения.

## Ресурсы API

- `/auth/`: Аутентификация
- `/users/`: Управление пользователями
- `/topics/`: Просмотр тем
- `/materials/`: Просмотр учебных материалов

## Как запустить проект:

Клонировать репозиторий и перейти в него в командной строке:

```
git clone https://github.com/Kirill374mansurov/NewStudygram.git
```

Запустить проект в Docker:

```
docker compose up --build
```
